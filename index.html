Resumen de las primeras 4 semanas

Semana 1: Introducción al Desarrollo Web

Resumen: Se presentaron los conceptos básicos de la web y el rol del desarrollador.

Taller: Conocimos las herramientas de trabajo y simulamos la estructura de un equipo de desarrollo.

Semana 2: Introducción al HTML

Resumen: Aprendimos a crear la estructura de una página web con HTML.

Taller: Creamos nuestras primeras páginas con títulos, párrafos, enlaces e imágenes.

Semana 3: Introducción al CSS

Resumen: Vimos cómo dar estilo a nuestras páginas usando reglas CSS.

Taller: Aplicamos colores, tipografías y espaciado a un portafolio personal.

Semana 4: Introducción al JavaScript (DOM, HTTP y Fetch)

Resumen: Aprendimos a usar JavaScript para interactuar con el contenido de la página.

Taller: Manipulamos el DOM, usamos eventos y trajimos datos desde una API con Fetch.
